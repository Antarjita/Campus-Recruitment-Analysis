Building the random forest model :

1.Initially we set the ntree=1 , this implies that a single decision tree is used to predict the status. Let us check the accuracy of this model and then proceed with the construction of multiple decision trees.

set.seed(80)
rfd <-randomForest(status~.,data=train, ntree=1,mtry=3,importance=TRUE)

  1. The error rate of a single decision tree is displayed above. Let us now increase the ntree value and find out the accuracy and the error rate of the random forest model.
